Malbum is a bit like a tiny, private Instagram. It consists of two parts: a server written in Clojure and an Android app written in Java. The server communicates with the app using JSON sent over a RESTful interface. For its web front-end, the server uses Foundation 6; its back-end is a Postgres database.

Through the web interface, users can register accounts, upload photos, comment on photos, browse albums and administrators can perform management tasks. Users can also browse albums and upload photos directly from their phones using the Android app.

Users can give photos custom names and descriptions. Commenting users can either be logged-in or anonymous.

I wrote Malbum in order to better understand building RESTful interfaces and image handling on Android. From the beginning the focus was more on functionality than aesthetics, and there hans't been much visual work done on the site or app.

You can find the server running live here. Feel free to make an account and upload some photos of your own to test it out. The Android app isn't on the Play store, but I can send you the .apk upon request (or just compile it yourself).

Check out the source code for the server and Android App on my Github!
